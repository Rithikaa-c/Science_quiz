Interactive Science Quiz Board

An interactive, touch-enabled display board designed to make science learning engaging, adaptive, and fun. This project offers over 1000 multiple-choice science questions categorized into three levels of difficulty: Beginner, Intermediate, and Advanced.

Key Features

Touch-Enabled Interface – Intuitive navigation for students of all ages.

Three Difficulty Levels – Beginner, Intermediate, Advanced to match learning pace.

Bilingual Support – Questions available in Tamil and English.

Achievement Rewards – A congratulatory message appears when a student answers three consecutive questions correctly, boosting confidence and motivation.

Instant Feedback – Immediate right/wrong responses to enhance learning.

Purpose

Traditional science education tools often lack interactivity and instant feedback, which can hinder engagement and deeper understanding. This project aims to provide an inclusive and adaptive platform that improves science literacy and fosters curiosity among students.

Technology Stack

Frontend: HTML, CSS, JavaScript

Data: JSON for question storage

Deployment: GitHub Pages
